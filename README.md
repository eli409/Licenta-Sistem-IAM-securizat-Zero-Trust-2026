# Licenta-Sistem-IAM-securizat-Zero-Trust-2026

Acest repository conține implementarea practică a lucrării de licență dezvoltată utilizând Windows 11, Ubuntu și Windows Server în VMware. Proiectul reprezintă o soluție hibridă de management al identității și accesului (IAM) fundamentată pe principiile arhitecturale Zero Trust, având ca obiectiv verificarea continuă și granulară a fiecărei cereri de acces.
